# Conversor de Unidades

Conversor de unidades no terminal feito em C#. Converte temperatura, distância e peso.

## Conversões disponíveis

- Temperatura: Celsius ↔ Fahrenheit
- Distância: Km ↔ Milhas
- Peso: Kg ↔ Libras

## Conceitos utilizados

- Loops (while)
- Condicionais (if/else)
- Entrada do usuário pelo console
- Operações matemáticas
